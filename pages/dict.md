- `dict` is like a collection of many `key`, `value` pairs
- ```python
  # create an empty dictionary
  students = {}
  
  # add something into dictionary
  students["Luffy"] = 3_000_000_000
  students["Zoro"] = 1_111_000_000
  
  print(type(students))
  print(students)
  ```
